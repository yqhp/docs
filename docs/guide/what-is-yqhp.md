# 简介

`yqhp` 是一个开源免费的分布式自动化测试系统，提供强大的在线交互式编程体验。开源地址: https://github.com/yqhp

<iframe src="//player.bilibili.com/player.html?bvid=BV1V14y1Q7Pd&page=1" allowfullscreen="true" width="100%" height="400px" />

## 自动化支持

- Android/iOS 自动化 (appium)
- Web 自动化 (selenium)
- 接口自动化 (rest-assured)
- ...
