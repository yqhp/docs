# 插件

## 上传插件

> `系统管理/插件管理`，以 `yqhp-appium` 插件为例

1. 下载 Appium 插件（见下方`插件列表`）
2. 新建插件 -> 名称: `yqhp-appium` -> 保存
3. 上传下载好的插件文件

## 启用插件

`系统管理/项目管理`进行插件设置，不同项目可以设置不同插件
